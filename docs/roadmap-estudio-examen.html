1. GET vs POST *** 1 hora Parcial 1 Susti

Por que es #1: Vino en AMBOS examenes, es el concepto MAS preguntado

Memorizar:

GET: - Datos en la URL (visible) - Limitado ~2000 caracteres - Cacheable / Idempotente - Para: busquedas, filtros - Ejemplo URL: /buscar?q=html&page=2 POST: - Datos en el body (oculto) - Sin limite de tamano - NO cacheable / NO idempotente - Para: login, registro, crear datos - Los datos NO aparecen en la URL

3. CSS Box Model *** 45 min Parcial 1 Susti

Memorizar el orden (de adentro hacia afuera):

1. CONTENT - width/height (el contenido) 2. PADDING - Espacio interior (transparente) 3. BORDER - Borde del elemento 4. MARGIN - Espacio exterior (transparente) Ancho total = width + padding + border + margin content-box - width NO incluye padding/border border-box - width SI incluye padding/border (MEJOR)

4. CSS Specificity *** 30 min Parcial 1 Susti

Orden de prioridad (menor a mayor):

1. Elementos (p, div) - (0,0,0,1) 2. Clases (.clase) - (0,0,1,0) 3. IDs (#id) - (0,1,0,0) 4. Inline (style="...") - (1,0,0,0) 5. !important - GANA SIEMPRE Ejemplo: #id .clase p - (0,1,1,1) = 1 ID + 1 clase + 1 elemento

Resumen de Acciones Prioritarias

Alta Prioridad (Para completar el curso)

  1. Python completo - Lenguajes/Python/
    • Glosario
    • 10 módulos
    • Ejercicios
    • CGI y web
  2. SQL/MySQL completo - Lenguajes/SQL/
    • Glosario
    • Módulos (SELECT, INSERT, JOINs, etc.)
    • Ejercicios
  3. Protocolos de Red - Nueva sección
    • HTTP/HTTPS
    • TCP/IP
    • DNS
    • GET/POST detallado

Media Prioridad

  1. Integración Python-MySQL
  2. Exámenes parciales (1 y 2)
  3. Depuración en JS (DevTools, console, debugger)
  4. AJAX clásico (XMLHttpRequest)

Baja Prioridad

  1. Conceptos de computación (Hardware, SO, UNIX)
  2. Editores de código
  3. WebAssembly
  4. Docker (ya en roadmap largo plazo)

Cronograma Sugerido

Semana 1-2: Python Básico

  • Crear glosario y módulos 1-5
  • Sintaxis, variables, estructuras de datos

Semana 3: Python Avanzado

  • Módulos 6-10
  • Funciones, clases, archivos, CGI

Semana 4: SQL Básico

  • Glosario y módulos
  • SELECT, INSERT, UPDATE, DELETE

Semana 5: SQL Avanzado + Integración

  • JOINs, subqueries
  • Python + MySQL

Semana 6: Protocolos y Exámenes

  • Sección de protocolos
  • Crear exámenes parciales y final

M√©todos para encontrar elementos

üìã Transformar: map, filter, reduce

M√©todos funcionales esenciales

const nums = [1, 2, 3, 4, 5]; // map - transformar cada elemento const dobles = nums.map(n => n * 2); // [2, 4, 6, 8, 10] // filter - filtrar elementos const pares = nums.filter(n => n % 2 === 0); // [2, 4] // reduce - reducir a un valor const suma = nums.reduce((acc, n) => acc + n, 0); // 15 // Encadenar m√©todos const resultado = nums .filter(n => n > 2) .map(n => n * 2) .reduce((acc, n) => acc + n, 0); // 24 // Ejemplos pr√°cticos const productos = [ { nombre: "A", precio: 100 }, { nombre: "B", precio: 200 } ]; // Total de precios const total = productos.reduce((acc, p) => acc + p.precio, 0); // Solo nombres const nombres = productos.map(p => p.nombre);
üí° Tip: map/filter/reduce no modifican el array original, retornan uno nuevo.

üìã Ordenar

Ordenar y revertir arrays

// sort modifica el array original const letras = ["c", "a", "b"]; letras.sort(); // ["a", "b", "c"] // N√∫meros (¬°cuidado!) const nums = [10, 2, 5, 1]; nums.sort(); // [1, 10, 2, 5] - orden alfab√©tico // Correcto para n√∫meros nums.sort((a, b) => a - b); // [1, 2, 5, 10] ascendente nums.sort((a, b) => b - a); // [10, 5, 2, 1] descendente // Ordenar objetos const usuarios = [ { nombre: "Zoe", edad: 25 }, { nombre: "Ana", edad: 30 } ]; usuarios.sort((a, b) => a.edad - b.edad); usuarios.sort((a, b) => a.nombre.localeCompare(b.nombre)); // reverse const arr = [1, 2, 3]; arr.reverse(); // [3, 2, 1]
‚ö†Ô∏è Cuidado: sort() y reverse() modifican el array original. Usa [...arr].sort() para no modificarlo.
